Who I am

I'm a Computer Engineer with professional experience at Indra, where I develop real-time software for mission-critical embedded Linux systems using Ada — an environment that has deeply shaped how I think about robustness, fault tolerance, and software reliability.

Outside of work, I've built three production-grade Java/Spring Boot projects focused on event-driven architectures. claims-event-sourcing is an insurance claims management system built with Event Sourcing and CQRS — every state change is stored as an immutable Kafka event, with full audit history and state machine validation. fraud-detection-service is a real-time fraud detection system using Kafka, Spring Security with JWT, and the Strategy Pattern. FinTrack is a microservices platform simulating a banking system with 4 independent services, Kafka, and Docker.

I'm looking for Java backend roles in international teams, bringing the discipline of safety-critical development into distributed systems. EU citizen, open to relocation.
